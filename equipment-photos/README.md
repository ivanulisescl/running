# Fotos de equipos

Guarda aquí la foto de cada equipo (zapatilla, etc.). La app las cargará automáticamente según el **nombre del archivo**.

## Nombre del archivo

El nombre debe ser el **slug** del nombre del equipo:

- Todo en **minúsculas**
- **Espacios** → guión `-`
- **Sin acentos** (á→a, é→e, í→i, ó→o, ú→u, ñ→n)
- Solo letras, números y guiones

**Ejemplos:**

| Nombre del equipo (en la app) | Nombre del archivo |
|-------------------------------|--------------------|
| Asics Gel Nimbus 25 Negras    | `asics-gel-nimbus-25-negras.jpg` |
| Asics Gel Nimbus 26 Azules    | `asics-gel-nimbus-26-azules.jpg` |
| Hokka Bondi 9 Grises          | `hokka-bondi-9-grises.jpg` |

## Formato y tamaño

- **Formato:** JPG (recomendado) o WebP.
- **Tamaño en pantalla:** 96×96 píxeles.  
- **Resolución del archivo:** 96×96 px o 192×192 px (para pantallas retina).
- **Peso orientativo:** 10–50 KB.

Puedes usar extensión `.jpg` o `.webp`; la app busca primero `.webp` y luego `.jpg`.
